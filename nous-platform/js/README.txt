NEURORDER CALENDAR ALPHA

Copy these into your Nous Platform:
- calendar.html -> project root
- css/calendar.css -> css/
- js/calendar.js -> js/

The app currently saves events in localStorage and uses Nous Core when available.
It does not yet write to the Supabase calendar_events table.

Required existing paths:
- assets/apps/calendar.png
- assets/images/default-profile.png
- js/supabase.js
- js/access.js
- js/nous-core.js

Link the Calendar icon in personal.html to calendar.html.
